# Dungeon-Escape
# UE5 Interaction Systems Prototype

This project is an **Unreal Engine 5 C++ prototype** demonstrating several interactive gameplay systems commonly used in puzzle and exploration levels.

# Implemented Mechanics

* Line Trace and Shape Trace based interaction for detecting objects in the environment
* Tag-based actor detection to identify interactable objects
* Actor pickup system for interacting with and carrying objects
* Lock mechanism where placing an item triggers movement of another actor
* Trigger and Mover components used to control dynamic actor movement
* Pressure plate system that activates another actor when stepped on or when an object is placed on it
* Spotlight feature used to illuminate specific elements in the level

# Tech Used

* Unreal Engine 5
* C++

This project focuses on building modular gameplay systems and understanding how actors communicate and interact within an Unreal Engine environment.
